### ‚ú® Core Features

-   **Intuitive Multi-Section Form**: Logically divided sections for personal info, summary, work experience, education, and skills make data entry a breeze.

### üì¶ Installation & Local Setup

To get this project running on your local machine, follow these simple steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-repo/ai-resume-builder.git
    cd ai-resume-builder
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    # or
    yarn dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) in your browser to see the application live.
